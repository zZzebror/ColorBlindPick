# ColorBlindPick
Vibe coded color scheme suggestor with images for model painting

## Usage

Generate a random color scheme:
```bash
python3 main.py
```

Generate a specific harmony type:
```bash
python3 main.py complementary
```

Available harmony types:
- monochromatic
- analogous
- complementary
- split_complementary
- triadic
- tetradic

## Output

The generator provides:
- Color palette (dominant, secondary, accent, neutral)
- Color theory explanation
- Painting technique suggestions
- Practical tips
- Hobby trivia
- Image prompt for visualization
